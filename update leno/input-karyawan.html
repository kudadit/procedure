Input Karyawan Baru

  1. Buka web LenO dan klik bagian SDM yang berada dibagian kiri website
  2. Klik karyawan dan klik + tambah pada halaman daftar karyawan untuk menginput karyawan baru
  3. Isi semua bagian sesuai dengan dokumen personal karyawan yang diterima
  4. Input dokumen KTP, KK, dan NPWP dengan format PDF
  5. Isi "2" untuk status vaksinasi, lalu pilih "vaksin program pemerintah" untuk program vaksinasi
  6. Klik data aktif dan simpan untuk mengaktifkan akun
  7. Apabila telah menerima dokumen mengenai pendidikan, pelatihan, dan dokumen lainnya, lengkapilah akun dengan:
    • Cari akun pada daftar karyawan.
    • Klik akun tersebut, lalu klik bagian history dan lengkapilah semua informasi yang telah diperoleh
